# Hisaho_HP

認定こども園 ひさほ保育園の静的Webサイトです。

## 構成

- `index.html` - トップページ、園について、保育の特色、食育、行事、入園案内
- `news.html` - お知らせ一覧と各ページへの導線
- `visit.html` - 園見学申込ページ
- `recruit.html` - 保育士募集ページ
- `styles.css` - ブランドカラー、レイアウト、アニメーション
- `script.js` - モバイルナビゲーション、スクロール表示
- `assets/` - パンフレットから抽出した写真・ロゴ素材

## 確認方法

ブラウザで `index.html` を開いて確認できます。

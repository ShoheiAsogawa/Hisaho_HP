# Hisaho_HP

認定こども園 ひさほ保育園の静的Webサイトです。

## 構成

- `index.html` - トップページ、園について、過ごし方、食育、年間行事、入園案内
- `news.html` - お知らせ一覧とInstagram導線
- `visit.html` - 園見学申請ページ
- `recruit.html` - 保育士募集ページ
- `styles.css` - ブランドカラー、レイアウト、アニメーション
- `script.js` - モバイルナビゲーション、スクロール表示、フォーム補助

## 確認方法

ブラウザで `index.html` を開いて確認できます。
